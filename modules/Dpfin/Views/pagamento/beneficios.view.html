A conversão de uma composição nada mais é que transformar os registros de uma composição em títulos. Ou seja, cada item listado na composição que foi importada para dentro do sistema de um software de terceiros (Nasajon Persona, TOTVs e etc) irá gerar um documento de pagamento para o sistema financeiro de forma independente. Em resumo, uma composição possibilita o envio de uma ordem de pagamentos em lote para o sistema financeiro.
